# مسار السيارات

موقع عربي ثابت عن السيارات، يضم مقالات عن شراء السيارات، الصيانة، توفير الوقود، التأمين، الإطارات، والسيارات العائلية.

## النشر

يمكن نشر الموقع عبر GitHub Pages من إعدادات المستودع:

Settings -> Pages -> Deploy from a branch -> main -> /root

بعد ربط دومين حقيقي، حدّث `sitemap.template.xml` بالدومين واحفظه باسم `sitemap.xml`.
