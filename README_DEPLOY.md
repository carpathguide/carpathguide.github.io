# مسار السيارات - خطوات النشر

هذا موقع ثابت يمكن نشره على GitHub Pages أو Netlify أو Vercel أو أي استضافة تدعم ملفات HTML.

## قبل التقديم على AdSense

1. انشر الموقع على دومين حقيقي.
2. اربط صفحة التواصل ببريد رسمي أو خدمة نماذج.
3. استبدل `https://your-domain.com` في `sitemap.template.xml` بالدومين الحقيقي ثم احفظه باسم `sitemap.xml`.
4. أضف رابط sitemap النهائي داخل `robots.txt`.
5. أضف مقالات أصلية جديدة بشكل منتظم.
6. بعد قبول AdSense، أضف ملف `ads.txt` بالقيمة التي يعطيك إياها Google.

## ملاحظة

لا ترسل بيانات حساب Google أو بيانات الدفع لأي شخص. خطوات AdSense الحساسة يجب تنفيذها من حسابك أنت.
